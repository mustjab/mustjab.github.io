About the Language Detection API

Key Features

  • Client-side Processing: All detection happens locally, protecting user privacy
  • Ranking Model: Returns multiple language candidates ranked by confidence
  • On-demand Download: The small detection model downloads automatically when needed
  • Wide Language Support: Supports detection of numerous languages

Use Cases

  • Determine language before translation
  • Load language-specific models for tasks like toxicity detection
  • Label content correctly in social networking sites
  • Adjust app interface based on user's language

Best Practices

  • Avoid very short phrases or single words for better accuracy
  • Use confidence thresholds to filter unreliable detections
  • Consider showing multiple candidates for ambiguous text
  • Test with your priority languages and set appropriate confidence levels

Browser Support

This API is currently available in Chrome 138+ with the built-in AI feature enabled.
